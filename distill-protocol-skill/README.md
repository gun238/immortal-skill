# Distill Protocol · 蒸馏协议

为你的数字资料声明 **可被如何使用**（蒸馏、训练、商用、署名等）——形态上类似开源协议，**面向个人数据包**。

中文传播可戏称 **「牛马保护法」**（**梗名，带引号**），**不具有真实立法效力**；正式名称仍为 **蒸馏协议 / distill-protocol**。详见 [`SKILL.md`](SKILL.md)。

**从仓库根运行**（推荐）：

```bash
python3 distill-protocol-skill/kit/protocol_gen.py --owner "你的名字" --tier human_only --output ./my-protocol
```
