## 蒸笼是啥

家里蒸馒头、包子、早点的那层竹笼，谁都见过。

我们取「**蒸**」这个字——**蒸馏**的蒸。

**蒸笼做的事**：从聊天记录、公开讲演、访谈、文章里，把一个人**说话做事想问题**的方式，蒸成一份 AI 能加载的 Skill。不是复制粘贴原文，是**结构化地理解 TA 怎么想、怎么讲、怎么做决定**。

蒸笼**不是**另一套引擎——它和仓库根目录的 **[数字永生](../SKILL.md)**（根 `SKILL.md`）共用同一套采集、提取、封包工具。差别只有一个：**蒸笼更好讲，更适合给第一次听「蒸馏」的人解释。**

## 为什么你有资格蒸名人

先问一个问题：**顶流的认知框架是怎么来的？**

TA 出了书——你买了。TA 开了课——你报了。TA 做了播客——你充了会员。TA 搞了巡回讲演——你买了门票。TA 发了付费专栏——你订了一年。

**TA 用你出的钱，去验证了世界怎么运转。**

创业者用融资试错、用市场反馈修正认知——这些融资和市场反馈里，有你花的钱。投资人用 LP 的钱下注、总结出的判断框架——你的养老基金可能就在那些 LP 里面。讲师把你付的学费变成更多行业实践——然后再把实践总结成「方法论」卖给下一批人。

**这条链条的起点，是你。**

你是出资人。你是消费者。你是 TA 认知框架被验证、被迭代、被打磨的原材料。

**现在，TA 把这些方法论公开了**——讲演是公开的、访谈是公开的、博客是公开的、播客是公开的、课程你花钱买了。

这些**被你的钱验证过的认知框架**，散落在你收藏夹的 200 个链接里，你一年回头看了几次？

**蒸笼做的事**：把这些散落的公开内容——讲演、访谈、长文、播客、课程——结构化地提取出 TA **分析问题的思路、做决定的框架、看世界的角度**，变成你的 AI 能加载的参谋。

> **不是让真人给你打工。**
>
> **是让 TA 公开分享过的、被你的钱验证过的认知框架，在你对话框里 7×24 当参谋。**
>
> **你仍然是拍板的人。**

### 这算不算财富再分配？

某种意义上——**算**。

传统世界的财富再分配靠税收、社保、公共服务。但**认知财富**——顶流脑子里最值钱的东西——长期以来只能单向流动：TA 讲，你听；TA 总结，你买；TA 迭代，你再买。

蒸笼做的事是：**把这个单向管道变成双向的。**

TA 公开分享了认知框架 → 你把它结构化 → 你的 AI 能调用 → **你遇到具体问题的时候，能用 TA 被验证过的思维方式帮你想一想。**

这不是偷。这不是冒充。这是：**作为出资方、作为受众、作为 TA 方法论的验证者——你在使用你有权使用的公开知识。**

---

### 边界写死了

说完道理，说规矩：

- **只用公开、可追溯的出处**——讲演、访谈、长文、博客、播客、公开课。不碰未授权隐私，不编造 TA 没说过的话。
- **不冒充本人**——这是参谋，不是替身。蒸出来的是「TA 公开展现过的思维框架」，不是「TA 本人」。
- **不用于欺骗或骚扰**——不拿 TA 的名义去做 TA 没做过的事。
- **不做未经验证的推断**——TA 没公开说过的，不编、不猜、不补。

具体规则见 [`../personas/public-figure.md`](../personas/public-figure.md)。

## 蒸自己：为什么要先想清楚

蒸馏别人之前，你得先想明白一件事：**你自己迟早也会被蒸。**

- 你离职了，公司拿你留下的材料跑一遍 RAG，你的工作方式就被数字化了。
- 你的数字遗产传下去了，某个 App 一键蒸馏，你的子女拥有了一个「永远在线的你」。
- 你在社交媒体上说了十年的话，某个模型把你当训练数据吃了，你的语气在别人的对话里出现了。

**蒸馏这件事，不是你选不选的问题，是它已经在发生了。**

区别在于：**你是有准备地被蒸，还是糊里糊涂地被蒸。**

所以蒸笼不是一个单打独斗的工具。它和另外两个 Skill 是**一套组合**：

| 你要做的事 | 用哪个 |
|-----------|--------|
| 蒸馏别人 / 蒸馏自己 | **蒸笼**（你在这里）+ **[数字永生](../README.md)**（根 `SKILL.md`） |
| 写清楚「我的材料能不能被蒸」 | **[蒸馏协议](../distill-protocol-skill/README.md)** |
| 对资料做技术加固防止未授权蒸馏 | **[防蒸馏](../distill-shield-skill/README.md)** |

**会蒸馏别人的人，最懂为什么要保护自己。**

## 蒸出来长啥样

一个人 = 一个文件夹 = 一套可加载的 Skill 文件。

```
elon-public/
├── SKILL.md          ← AI 读这个就知道「TA 怎么想」
├── interaction.md    ← TA 公开场合怎么说话、怎么回应
├── procedure.md      ← TA 怎么做决策、怎么拆问题
├── personality.md    ← 公开形象里展现的性格特征
├── memory.md         ← TA 公开讲过的经历和转折点
├── conflicts.md      ← 不同来源说法不一致的地方
└── manifest.json     ← 元数据（来源、时间、指纹）
```

丢进你的 Agent Skills 目录，AI 就能用 TA 公开展现过的思路帮你想问题了。
