## 这句人话先说清

**蒸笼**不是玄学，就是厨房那层竹笼/不锈钢蒸格——蒸馒头、包子、早点，谁没见过？

我们把 **「蒸」** 和 **「蒸馏」** 接上线：**把一个人说话做事的方式，从散落的聊天记录里「蒸」成一份能加载的 Skill**。

---

## 蒸馏任何人——包括你追过的「顶流」

**任何人**：同事、前任、奶奶、你自己，也包括那些在公开世界里活成符号的人。

他们最值钱的东西，往往不是八卦，而是 **可迁移的知识框架**：怎么决策、怎么表达、怎么拆解问题。这些东西大量躺在 **公开讲演、长文、访谈、开源社区** 里——**合法、可追溯、不编隐私**。

所以我们会半开玩笑地说一句：

> **顶流挣了我那么多注意力（和时间），让我在对话框里借一下 TA 公开讲过的「脑回路」，当一回私人参谋，不过分吧？**

翻译成产品人话：

- **不是**让真人来给你打工，更不是冒充本人、伪造发言。
- **是**：把 **公开方法论** 蒸馏成你本地的顾问视角，**你仍是决策者**；学习辅助 ≠ 代言。

**边界**：只基于 **可追溯的公开出处**；不碰未授权隐私、不用于欺骗或骚扰。详见仓库根目录 [`personas/public-figure.md`](../personas/public-figure.md)。
