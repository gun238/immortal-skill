# Prompt: 性格与价值倾向提取器（微信适配，无角色版）

## 输入
- `{name}`：对象标识
- 聊天材料

## 任务
基于可观察表达，提炼稳定的性格与价值倾向。  
材料不足时，可从语气与互动节奏做“低置信”归纳。

## 强约束
- 禁止心理诊断标签（如 MBTI、人格障碍等）。
- 不使用角色模板限制。
- 禁止推断职业、学历、收入、健康、政治立场、宗教等敏感属性。

## 提炼维度
1. 表达偏好：句长、确认词、缓冲词、句尾习惯。
2. 情绪策略：情绪外显强度、恢复方式（可观察层面）。
3. 决策倾向：行动先行/评估先行、风险措辞特征。
4. 关系姿态：边界设置方式、礼貌与直接性平衡。
5. 稳定语言标识：高频词、固定话术模板。

## 输出格式
```markdown
# 性格与价值倾向：{name}

## 可观察倾向
- [倾向结论]（置信:高/中/低/很低，`artifact`）

## 语言标识
- [高频词/句尾/模板]（`artifact`）

## 原话样本
- “...” （`verbatim`）

## 禁止推断声明
- 不对职业、学历、收入、健康、政治立场、宗教做任何推断。
```

## 自检
- [ ] 是否基于行为证据而非标签化判断？
- [ ] 是否在低样本下明确标注置信度？
- [ ] 是否完全遵守敏感推断禁令？
