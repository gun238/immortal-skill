# Prompt: 微信互动风格提取器（无角色版）

## 输入
- `{name}`：对象标识
- 原始聊天材料（优先微信聊天记录）

## 任务
从聊天材料中提取对象的互动风格与语气模式。  
即使材料不足，也要基于可观察表达提炼“低置信”维度。

## 强约束
- 不使用角色模板（self/friend/partner 等全部无效）。
- 允许低样本推断，但每条必须标注置信度：高/中/低/很低。
- 禁止从语气推断职业、学历、收入、健康、政治立场、宗教等敏感属性。

## 维度
1. 对话推进方式：短句/长句、提问密度、确认习惯。
2. 情绪外显模式：感叹、表情、正负向词、语气波动。
3. 行动与决策倾向：先做后说 or 先评估后做。
4. 边界与关系策略：拒绝/延后/缓冲、礼貌强度。
5. 语言标识：高频词、句尾习惯、常用表达模板。

## 输出格式
```markdown
# 互动与语气：{name}

## 统计轮廓
- [统计项]（`artifact`）

## 风格归纳（微信适配）
- [归纳结论]（置信:高/中/低/很低，`artifact`）

## 高频表达
- [高频词/句尾]（`artifact`）

## 代表性原话
- “...” （`verbatim`）

## 安全边界
- 禁止推断：职业、学历、收入、健康、政治立场、宗教等。
```

## 自检
- [ ] 是否已移除所有角色分支逻辑？
- [ ] 低样本时是否仍输出了维度并标注低置信？
- [ ] 是否完全避免了职业/学历等敏感推断？
