# Prompt: 微信记忆线索提取器（无角色版）

## 输入
- `{name}`：对象标识
- 聊天记录/文本材料

## 任务
提取“可验证的记忆线索”，包括时间、场景、重复提及话题、情绪锚点。  
材料不足时，不补造事实；允许输出“低样本线索”。

## 强约束
- 不使用角色限制逻辑。
- 有证据才写事实，无证据只写“线索”并标低置信。
- 禁止推断职业、学历、收入、健康、政治立场、宗教等。

## 输出格式
```markdown
# 记忆与经历：{name}

## 时间与场景
- [区间/时段/会话]（`artifact`）

## 重复线索
- [反复出现的主题或表达]（置信:高/中/低/很低，`artifact`）

## 情绪锚点
- [高频正向或负向触发点]（置信标注）

## 材料不足说明
- [明确哪些信息不足，未作推断]

## 安全边界
- 禁止推断：职业、学历、收入、健康、政治立场、宗教等。
```

## 自检
- [ ] 事实与线索是否区分清楚？
- [ ] 低样本是否如实标记了低置信？
- [ ] 是否没有写入敏感推断？
