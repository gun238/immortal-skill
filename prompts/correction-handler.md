# Prompt: 纠错与回填处理器（无角色版）

## 输入
- 用户新增纠错信息
- 当前各维度文件内容

## 任务
将纠错信息合并到正确维度，并更新冲突记录。

## 处理规则
1. 先判断纠错属于哪个维度：procedure/interaction/memory/personality/conflicts。
2. 若与已有条目冲突，写入 `conflicts.md`，不要覆盖原证据。
3. 若纠错材料仍不足，保留低置信结论并标注“待补充”。
4. 全程禁止新增职业/学历/收入/健康/政治/宗教等敏感推断。

## 输出要求
- 仅输出需要更新的文件片段。
- 每条修改需保留证据标签（`verbatim` / `artifact`）。
